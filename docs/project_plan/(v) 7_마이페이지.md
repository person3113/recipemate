### 🎬 사용자 액션 흐름

### 1) [프로필 편집] 버튼 클릭

- **이동**: 프로필 편집 페이지
- **편집 가능 항목**:
    - 프로필 이미지 (업로드/변경/삭제)
    - 닉네임 (중복 확인 필요)
    - 비밀번호 변경 (현재 비밀번호 확인 후)
    - 전화번호
    - ~~소개 (선택, 최대 200자)~~
- **[저장] 버튼**:
    - 유효성 검사
    - 저장 후 "프로필이 수정되었습니다" 알림
    - 마이페이지로 돌아가기

### 2) 매너온도 확인

- **색상**:
    - 36.5 이상: 초록색
    - 33.0~36.4: 노란색
    - 33.0 미만: 빨간색

### 3) 뱃지 확인

- **표시**: 획득한 뱃지 아이콘 나열
