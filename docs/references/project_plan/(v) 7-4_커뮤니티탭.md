## 👤 7-4. [커뮤니티] 탭

### 📍 화면 구성

```
┌───────────────────────────────────────────────┐
│  [내가 쓴 글] [내가 쓴 댓글] [좋아요한 글]       │
├───────────────────────────────────────────────┤
│  게시글 리스트 (커뮤니티 목록과 동일 형식)       │
│  ┌───────────────────────────────────────┐    │
│  │ [꿀팁] 식재료 보관법 꿀팁!              │    │
│  │ 2025-10-25  |  👁️ 156  💬 12  ❤️ 23    │    │
│  │                                       │    │
│  └───────────────────────────────────────┘    │
│  ...                                          │
└───────────────────────────────────────────────┘

```

### 🎬 사용자 액션 흐름

### 1) 탭 전환

- **[내가 쓴 글]**: 내가 작성한 게시글 목록
- **[내가 쓴 댓글]**: 내가 작성한 댓글 목록 (게시글 제목 + 댓글 내용)
- **[좋아요한 글]**: 내가 좋아요한 게시글 목록

### 2) [보기] 버튼

- **클릭**: 해당 게시글 상세 페이지로 이동
