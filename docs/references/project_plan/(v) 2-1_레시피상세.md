### 3) 진행 중인 공구 확인

- **표시 조건**: 이 레시피 기반으로 만들어진 모집중 공구 있으면 표시
- **[공구 상세 보기] 클릭**: 해당 공구 상세 페이지로 이동
- **없으면**: 이 섹션 자체가 숨김

### 4) [이 재료로 공구 만들기] 버튼 클릭 ⭐핵심⭐

- **로그인 확인**:
    - 비로그인 → 로그인 페이지로 리다이렉트
    - 로그인됨 → 공구 작성 페이지로 이동
- **데이터 전달**:
    - 레시피 ID
    - 체크된 재료 목록 (strIngredient)
    - 분량 목록 (strMeasure, 인원 설정 반영)
    - 레시피명
    - 레시피 이미지
- **자동 입력**: 공구 작성 페이지에서 아래 항목 자동 채워짐
    - 제목: "[레시피명] 재료 공동구매"
    - 내용: "이 레시피를 만들기 위한 재료 공구입니다\n\n[재료 목록 자동 나열]"
    - 이미지: 레시피 이미지 자동 첨부
    - 카테고리: 레시피 카테고리 자동 선택
