### 🎬 사용자 액션 흐름

### 1) 이미지 갤러리 (있는 경우)

- **슬라이드**: 좌우 화살표로 이미지 전환
- **확대**: 이미지 클릭 시 전체 화면 모달

- ~~**표시**: 공구 제목 + 아이콘~~

~~### 3) 태그 클릭~~

- ~~**클릭**: 해당 태그가 달린 게시글 목록으로 이동~~
- ~~**표시**: "# [태그명]" 검색 결과~~

### 4) [좋아요 ❤️] 버튼

- **로그인 확인**: 비로그인 → 로그인 페이지
- **클릭**:
    - 빈 하트 → 꽉 찬 하트 (좋아요 +1)
    - 꽉 찬 하트 → 빈 하트 (좋아요 -1)
    - 숫자 실시간 업데이트

### 5) [공유 📤] 버튼

- **공유 옵션**: 링크 복사, 카카오톡, 이메일 (공구 상세와 동일)

### 6) 작성자 전용 기능

- **[수정]**:
    - 클릭 시 글쓰기 페이지 (수정 모드)
    - 기존 내용 자동 입력
- **[삭제]**:
    - "정말 삭제하시겠습니까?" 확인
    - [확인] → 삭제 후 커뮤니티 목록으로 이동

### 7) 댓글 작성

- **입력**: 텍스트 입력 (최대 500자)
- **[등록] 클릭**:
    - 댓글 즉시 추가
    - "댓글이 등록되었습니다" 알림
    - 알림 생성: 작성자에게 "OOO님이 댓글을 남겼습니다" 알림

### 8) 댓글 액션

- **[답글]**:
    - 클릭 시 대댓글 입력창 표시
    - 입력 후 [등록] → 인덴트되어 대댓글 표시
- **[좋아요]**:
    - 댓글 좋아요 +1/-1
- **[수정]/[삭제]** (본인 댓글만):
    - 수정: 인라인 편집 모드
    - 삭제: 확인 후 삭제
