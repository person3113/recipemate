
## 👤 7-2. [참여 공구] 탭

### 📍 화면 구성

```
┌───────────────────────────────────────────────┐
│  내가 참여한 공구 (8)                           │
│  [참여중] [완료] [취소] 필터                    │
├───────────────────────────────────────────────┤
│  ┌───────────────────────────────────────┐    │
│  │ 🍝 까르보나라 재료 공구 (홍길동)        │    │
│  │ 상태: 참여중  |  3/5명  |  D-2          │    │
│  │ [공구 보기] [참여 취소]                 │    │
│  └───────────────────────────────────────┘    │
│  ┌───────────────────────────────────────┐    │
│  │ 🍗 치킨 공동구매 (김철수)               │    │
│  │ 상태: 완료 ✅  |  10/10명               │    │
│  │ [공구 보기] [후기 작성]                 │    │
│  └───────────────────────────────────────┘    │
│  ...                                          │
└───────────────────────────────────────────────┘

```

### 🎬 사용자 액션 흐름

### 1) 필터 선택

- **[참여중]**: 아직 진행 중인 공구
- **[완료]**: 마감/완료된 공구
- **[취소]**: 내가 참여 취소한 공구

### 2) [공구 보기] 버튼

- **클릭**: 해당 공구 상세 페이지로 이동

### 3) [참여 취소] 버튼 (참여중만)

- **확인**: "참여를 취소하시겠습니까?"
- **취소**: 참여 정보 삭제, 주최자에게 알림

### 4) [후기 작성] 버튼 (완료만)

- **클릭**: 후기 작성 팝업
- **입력 항목**:
    - 별점 (1~5개, 필수)
    - 텍스트 후기 (선택, 최대 500자)
- **[등록] 클릭**:
    - 후기 저장
    - 주최자 매너온도 반영:
        - 5점: +0.5°C
        - 4점: +0.3°C
        - 3점: 변동 없음
        - 2점: -1.0°C
        - 1점: -2.0°C
    - 알림: 주최자에게 "OOO님이 후기를 남겼습니다" 알림
    - 포인트: +5P 획득
