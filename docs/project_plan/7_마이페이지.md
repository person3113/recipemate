## 👤 7. 마이페이지

### 📍 화면 구성

```
┌───────────────────────────────────────────────┐
│  마이페이지 👤                                  │
│  [프로필 편집] 버튼                             │
├───────────────────────────────────────────────┤
│  ┌─────────────────────────────────────┐      │
│  │ [프로필 이미지]                        │      │
│  │                                       │      │
│  │ 홍길동                                │      │
│  │ ⭐ 매너온도: 37.2°C 🌡️               │      │
│  │ 🎖️ 뱃지: [첫 공구 주최자] [친절한 이웃] │      │
│  │ 💰 포인트: 150P                       │      │
│  └─────────────────────────────────────┘      │
├───────────────────────────────────────────────┤
│  [내 공구] [참여 공구] [찜 목록] [커뮤니티]     │
│  [알림] [설정] 탭                              │
└───────────────────────────────────────────────┘

```

### 🎬 사용자 액션 흐름

### 1) [프로필 편집] 버튼 클릭

- **이동**: 프로필 편집 페이지
- **편집 가능 항목**:
    - 프로필 이미지 (업로드/변경/삭제)
    - 닉네임 (중복 확인 필요)
    - 비밀번호 변경 (현재 비밀번호 확인 후)
    - 전화번호
    - 소개 (선택, 최대 200자)
- **[저장] 버튼**:
    - 유효성 검사
    - 저장 후 "프로필이 수정되었습니다" 알림
    - 마이페이지로 돌아가기

### 2) 매너온도 확인

- **hover**: 툴팁 표시
    - "매너온도는 36.5°C에서 시작합니다"
    - "좋은 후기: +0.5°C"
    - "나쁜 후기: -2.0°C"
    - "신고 누적: -3.0°C"
- **색상**:
    - 36.5 이상: 초록색
    - 33.0~36.4: 노란색
    - 33.0 미만: 빨간색

### 3) 뱃지 확인

- **표시**: 획득한 뱃지 아이콘 나열
- **hover**: 뱃지명 + 획득 조건 툴팁
- **예시 뱃지**:
    - 🎖️ 첫 공구 주최자 (첫 공구 등록 시)
    - 🤝 친절한 이웃 (후기 평균 4.5 이상)
    - 🏆 파워 참여자 (공구 10회 참여)
    - 🍳 레시피 마스터 (레시피 기반 공구 5회 주최)

### 4) 포인트 확인

- **획득 방법**:
    - 공구 참여: +10P
    - 후기 작성: +5P
    - 커뮤니티 글 작성: +3P
    - 댓글 작성: +1P
- **사용 (선택 기능)**:
    - 향후 확장: 포인트로 배너 광고, 프로필 꾸미기 등
