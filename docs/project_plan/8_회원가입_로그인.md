
## 🔐 8. 회원가입/로그인

### 📍 회원가입 페이지

```
┌───────────────────────────────────────────────┐
│  회원가입 📝                                    │
├───────────────────────────────────────────────┤
│  이메일 *: [                    ] [중복확인]   │
│                                                │
│  비밀번호 *: [                              ]  │
│  비밀번호 확인 *: [                         ]  │
│  (8자 이상, 영문+숫자+특수문자)                 │
│                                                │
│  닉네임 *: [                    ] [중복확인]   │
│                                                │
│  전화번호 *: [010-    -    ]                   │
│                                                │
│  [이용약관 동의] ☑️ (필수)                      │
│  [개인정보 처리방침] ☑️ (필수)                  │
│  [마케팅 수신 동의] ☐ (선택)                    │
├───────────────────────────────────────────────┤
│  [가입하기] 버튼                                │
└───────────────────────────────────────────────┘

```

### 🎬 사용자 액션 흐름

### 1) 이메일 입력 + [중복확인]

- **입력**: 이메일 형식 유효성 검사 (실시간)
- **[중복확인] 클릭**:
    - 서버에 중복 여부 확인
    - "사용 가능한 이메일입니다" or "이미 사용 중인 이메일입니다"

### 2) 비밀번호 입력

- **실시간 검사**:
    - 8자 이상
    - 영문 포함
    - 숫자 포함
    - 특수문자 포함
- **색상 표시**:
    - 조건 충족 시 초록색 체크
    - 미충족 시 빨간색 X
- **비밀번호 확인**: 일치 여부 실시간 검사

### 3) 닉네임 입력 + [중복확인]

- **유효성**: 2~10자, 한글/영문/숫자
- **[중복확인] 클릭**:
    - "사용 가능한 닉네임입니다" or "이미 사용 중입니다"

### 4) 전화번호 입력

- **자동 하이픈**: 입력 시 자동으로 010-xxxx-xxxx 형식
- **숫자만 입력**: 문자 입력 시 자동 제거

### 5) 약관 동의

- **필수 약관**: 체크 안 하면 가입 불가
- **약관 보기**: 클릭 시 약관 내용 팝업

### 6) [가입하기] 버튼

- **유효성 최종 검사**:
    - 모든 필수 항목 입력
    - 이메일/닉네임 중복 확인 완료
    - 비밀번호 조건 충족
    - 필수 약관 동의
- **성공**:
    - 회원 정보 DB 저장
    - "회원가입이 완료되었습니다!" 알림
    - 자동 로그인 + 홈으로 이동
- **실패**:
    - 오류 필드에 빨간색 표시 + 오류 메시지

### 📍 로그인 페이지

```
┌───────────────────────────────────────────────┐
│  로그인 🔐                                      │
├───────────────────────────────────────────────┤
│  이메일: [                                  ]  │
│                                                │
│  비밀번호: [                                ]  │
│                                                │
│  ☐ 로그인 상태 유지                            │
│                                                │
│  [로그인] 버튼                                  │
├───────────────────────────────────────────────┤
│  [비밀번호 찾기] | [회원가입]                   │
└───────────────────────────────────────────────┘

```

### 🎬 사용자 액션 흐름

### 1) 이메일/비밀번호 입력

- **Enter 키**: 입력 후 Enter 시 로그인 실행

### 2) [로그인 상태 유지] 체크박스

- **체크 시**: 세션 유지 기간 7일
- **미체크 시**: 브라우저 종료 시 로그아웃

### 3) [로그인] 버튼 클릭

- **유효성 검사**:
    - 이메일/비밀번호 입력 확인
    - 이메일 형식 검사
- **서버 확인**:
    - 이메일 존재 여부 확인
    - 비밀번호 일치 여부 확인
- **성공**:
    - 세션 생성
    - "로그인되었습니다" 알림
    - 이전 페이지 또는 홈으로 이동
- **실패**:
    - "이메일 또는 비밀번호가 일치하지 않습니다" 오류 메시지
    - 5회 이상 실패 시: "5회 이상 실패했습니다. 10분 후 다시 시도해주세요" (계정 보호)

### 4) [비밀번호 찾기] 클릭

- **이동**: 비밀번호 찾기 페이지
- **입력**: 가입한 이메일 주소
- **[인증 메일 발송] 클릭**:
    - 해당 이메일로 인증 링크 발송
    - "인증 메일이 발송되었습니다" 알림
    - 이메일 링크 클릭 → 비밀번호 재설정 페이지
- **비밀번호 재설정**:
    - 새 비밀번호 입력 (조건 동일)
    - 새 비밀번호 확인
    - [변경] 클릭 → "비밀번호가 변경되었습니다" → 로그인 페이지

### 5) [회원가입] 클릭

- **이동**: 회원가입 페이지
