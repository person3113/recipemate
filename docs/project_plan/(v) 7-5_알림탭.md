```
┌───────────────────────────────────────────────┐
│  알림 🔔 (15)                                   │
│  [전체 읽음] [전체 삭제] 버튼                   │
├───────────────────────────────────────────────┤
│  ┌───────────────────────────────────────┐    │
│  │ 🔴 [새 알림] 김철수님이 참여했습니다    │    │
│  │ 까르보나라 재료 공구  |  5분 전         │    │
│  │ [공구 보기] [읽음]                      │    │
│  └───────────────────────────────────────┘    │
│  ┌───────────────────────────────────────┐    │

```

### 🎬 사용자 액션 흐름

### 1) 알림 종류

- **공구 관련**:
    - "OOO님이 참여했습니다"
    - "OOO님이 참여를 취소했습니다"
    - "공구가 마감되었습니다"
    - "공구가 목표 인원에 도달했습니다"
- **커뮤니티 관련**:
    - "OOO님이 댓글을 남겼습니다"
    - "OOO님이 대댓글을 남겼습니다"
    - "OOO님이 내 글을 좋아합니다"
- **후기 관련**:
    - "OOO님이 후기를 남겼습니다"

### 2) 알림 클릭

- **읽지 않음 (🔴)**: 클릭 시 읽음 처리 + 해당 페이지로 이동
- **읽음 (⚪)**: 클릭 시 해당 페이지로 이동

### 3) [전체 읽음] 버튼

- **클릭**: 모든 알림을 읽음 처리

### 4) [전체 삭제] 버튼

- **확인**: "모든 알림을 삭제하시겠습니까?"
- **삭제**: 모든 알림 삭제

### 5) 개별 알림 [읽음]/[삭제] 버튼

- **[읽음]**: 해당 알림만 읽음 처리
- **[삭제]**: 해당 알림만 삭제
