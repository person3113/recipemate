### 🎬 사용자 액션 흐름

### 1) 탭 전환

- **[꿀팁 공유]**: 식재료 보관, 조리 팁, 절약 팁 등
- **[자유 게시판]**: 자유로운 주제
- **[레시피 후기]**: 공구 후 실제 요리 후기 (사진 포함)

### 2) 검색 기능

- **검색창 입력**: 키워드 입력 (제목 + 내용 검색)
- **Enter 또는 검색 버튼**: 검색 결과 표시
- **검색 범위**: 현재 선택된 탭 내에서만 검색

### 3) [글쓰기] 버튼 클릭

- **로그인 확인**: 비로그인 → 로그인 페이지
- **이동**: 커뮤니티 글쓰기 페이지로 이동

### 4) 정렬 변경

- **최신순**: 작성일 기준 최신순
- **인기순**: 좋아요(❤️) 개수 많은 순
- **댓글많은순**: 댓글 수 많은 순

### 5) 게시글 카드 클릭

- **클릭**: 게시글 상세 페이지로 이동
- **표시 정보**:
    - 카테고리 태그 ([꿀팁], [자유], [후기])
    - 제목
    - 작성자 닉네임
    - 작성일
    - 조회수 (👁️)
    - 댓글 수 (💬)
    - 좋아요 수 (❤️)
    - 별점 (후기 게시글만)
