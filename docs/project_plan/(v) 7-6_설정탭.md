
## 👤 7-6. [설정] 탭

### 📍 화면 구성

```
┌───────────────────────────────────────────────┐
│  설정 ⚙️                                       │
├───────────────────────────────────────────────┤
│  📧 이메일: hong@example.com                  │
│  📱 전화번호: 010-1234-5678  [변경]            │
│                                                │
│  🔔 알림 설정:                                 │
│  ☑️ 공구 참여 알림                             │
│  ☑️ 댓글/대댓글 알림                           │
│  ☐ 마케팅 알림 (선택)                          │
│                                                │
│  🔒 개인정보:                                  │
│  [비밀번호 변경]                               │
│  [계정 탈퇴]                                   │
├───────────────────────────────────────────────┤
│  [저장] 버튼                                   │
└───────────────────────────────────────────────┘

```

### 🎬 사용자 액션 흐름

### 1) [전화번호 변경] 버튼

- **클릭**: 전화번호 입력 필드로 변경
- **입력**: 새 전화번호 입력
- **[저장]**: 변경 후 "전화번호가 변경되었습니다" 알림

### 2) 알림 설정 토글

- **체크/해제**: 각 알림 유형별 on/off
- **즉시 반영**: 체크 상태가 서버에 자동 저장

### 3) [비밀번호 변경] 버튼

- **클릭**: 비밀번호 변경 팝업
- **입력 항목**:
    - 현재 비밀번호 (확인용)
    - 새 비밀번호
    - 새 비밀번호 확인
- **[변경] 클릭**:
    - 현재 비밀번호 일치 확인
    - 변경 성공 시 "비밀번호가 변경되었습니다" 알림

### 4) [계정 탈퇴] 버튼

- **클릭**: 경고 팝업
    - "정말 탈퇴하시겠습니까?"
    - "진행 중인 공구가 있으면 탈퇴할 수 없습니다"
    - "모든 데이터가 삭제됩니다"
- **비밀번호 확인**: 본인 확인용 비밀번호 입력
- **[탈퇴] 클릭**:
    - 진행 중인 공구 확인
    - 있으면 "진행 중인 공구를 먼저 종료해주세요"
    - 없으면 계정 비활성화 
    - 로그아웃 + 메인 페이지로 이동
