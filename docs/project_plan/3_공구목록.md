### 🎬 사용자 액션 흐름

### 1) 검색 기능

- **입력**: 검색창에 키워드 입력
    - 예: "파스타", "치킨", "강남"
- **검색 범위**: 공구 제목 + 내용 + 재료명
- **Enter 또는 검색 버튼**: 검색 결과 표시
- **자동완성**: 입력 중 인기 검색어 드롭다운 표시 (선택)

### 2) 카테고리 필터

- **선택**: [식재료], [간식], [음료], [조리도구] 등
- **중복 선택**: 가능 (체크박스 방식)
- **[전체] 선택**: 모든 필터 해제
- **즉시 반영**: 선택 즉시 목록 갱신

### 3) 상태 필터

- **[전체]**: 모든 상태 표시
- **[모집중]**: 마감일 지나지 않고 목표 인원 미달성
- **[마감임박]**: D-1, D-2 공구만
- **[종료]**: 마감일 지났거나 목표 인원 달성

### 4) 정렬 변경

- **최신순**: 등록일 최신순
- **마감임박순**: 마감일 가까운 순 (D-1 → D-2 → D-3...)
- **가격순**: 1인당 가격 낮은 순
- **참여자순**: 현재 참여 인원 많은 순

### 5) 레시피 기반 공구만 보기

- **체크박스**: 체크 시 레시피 기반 공구만 필터링
- **표시**: 카드에 🍳 아이콘 있는 공구만
- **해제**: 전체 공구 표시

### 6) 공구 카드 클릭

- **카드 영역**: 클릭 시 공구 상세 페이지로 이동
- **카드 정보**:
    - 대표 이미지
    - 제목 (🍳 아이콘: 레시피 기반 표시)
    - 1인당 가격
    - 진행률 (현재/목표 인원, 퍼센트)
    - 마감일 (D-day, D-1이면 🔥 표시)
    - 수령 방법 (직거래 장소 또는 "택배 가능")
    - 진행률 색상바 (60% 미만=회색, 60-99%=파랑, 100%=초록)

### 7) 진행률 색상 의미

- **0-59%**: 회색 (💚) - 모집 초반
- **60-99%**: 파랑 (💙) - 모집 진행 중
- **100%**: 초록 (🧡) - 목표 달성 (모집 마감)
