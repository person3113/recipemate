### 🎬 사용자 액션 흐름

### 1) 레시피 검색

- **입력**: 검색창에 키워드 입력
    - 예: "파스타", "닭가슴살", "간단한 요리"
- **검색 방식**:
    - 레시피명 검색: TheMealDB `search.php?s=파스타`
    - 재료명 검색: TheMealDB `filter.php?i=chicken`
    - 한글 검색 시: 식품안전나라 API 병행 검색
- **결과 표시**: 일치하는 레시피 카드 목록
- **결과 없음**: "검색 결과가 없습니다" 메시지 + 랜덤 추천 3개

### 2) 카테고리 필터 선택

- **버튼 클릭**: [한식], [양식], [중식], [일식], [디저트]
- **동작**:
    - TheMealDB: `categories.php` 활용
    - 식품안전나라: 해당 카테고리만 필터링
- **UI 변화**: 선택된 버튼 하이라이트 (색상 변경)
- **중복 선택**: 불가능 (라디오 버튼 방식)

### 3) 정렬 변경

- **드롭다운 선택**:
    - 최신순: 최근 추가된 레시피 (캐시 시점 기준)
    - 인기순: 해당 레시피 기반 공구 개수 많은 순
    - 간단한순: 재료 개수 적은 순 (strIngredient 카운트)
    - 이름순: 가나다/ABC 순
- **즉시 반영**: 선택 즉시 목록 재정렬

### 4) 레시피 카드 클릭

- **동작**: 카드 전체 영역 클릭
- **이동**: 레시피 상세 페이지로 이동
- **카드 정보**:
    - 대표 이미지 (strMealThumb)
    - 레시피명 (strMeal 또는 번역된 한글명)
    - 난이도 (별점, 재료 개수로 추정: 5개 이하=하, 10개 이하=중, 그 이상=상)
    - 예상 조리 시간 (기본값 표시, API에 없으면 "약 30분")
    - 🍳 아이콘 (레시피 연동 표시)

### 5) 더보기/페이지네이션

- **더보기 버튼**: 클릭 시 다음 12개 레시피 추가 로드 (무한 스크롤 방식)
- **페이지네이션**: 번호 클릭 시 해당 페이지로 이동 (한 페이지 12개)
