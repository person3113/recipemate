
## 🏠 1. 홈 화면

### 📍 화면 구성

```
┌───────────────────────────────────────────────┐
│          🔥 인기 레시피 캐러셀                   │
│     [← 🍝파스타] [🍛카레] [🥗샐러드 →]          │
├───────────────────────────────────────────────┤
│          ⏰ 마감 임박 공구 배너                  │
│  "🔥 까르보나라 재료 공구 D-1! 서둘러 참여하세요"  │
├───────────────────────────────────────────────┤
│                카테고리별 공구                   │
│  [한식] [양식] [중식] [일식] [디저트] [간식]     │
├───────────────────────────────────────────────┤
│  공구 카드 그리드 (3열)                         │
│  ┌─────┐ ┌─────┐ ┌─────┐                     │
│  │공구1 │ │공구2 │ │공구3 │                     │
│  └─────┘ └─────┘ └─────┘                     │
│  ┌─────┐ ┌─────┐ ┌─────┐                     │
│  │공구4 │ │공구5 │ │공구6 │                     │
│  └─────┘ └─────┘ └─────┘                     │
├───────────────────────────────────────────────┤
│          💡 오늘의 추천 레시피                   │
│         [랜덤 레시피 3개 표시]                   │
└───────────────────────────────────────────────┘

```

### 🎬 사용자 액션 흐름

### 1) 인기 레시피 캐러셀 클릭

- **동작**: 캐러셀 레시피 이미지/제목 클릭
- **이동**: 해당 레시피 상세 페이지로 이동
- **추가**: 좌우 화살표로 슬라이드 전환 (자동 슬라이드 5초 간격)

### 2) 마감 임박 배너 클릭

- **동작**: 배너 전체 영역 클릭
- **이동**: 해당 공구 상세 페이지로 이동
- **표시**: 마감 D-1, D-2 공구만 순환 표시

### 3) 카테고리 버튼 클릭

- **동작**: [한식], [양식] 등 카테고리 버튼 클릭
- **이동**: 공구 목록 페이지로 이동 (해당 카테고리 필터 적용됨)
- **표시**: 선택된 카테고리만 공구 표시

### 4) 공구 카드 클릭

- **동작**: 공구 카드 전체 영역 클릭
- **이동**: 해당 공구 상세 페이지로 이동
- **정보 표시**:
    - 이미지 (대표 이미지 1장)
    - 제목 (최대 30자)
    - 1인당 가격
    - 진행률 바 (현재/목표 인원)
    - 상태 태그 (모집중/마감임박/마감)
    - 마감일 (D-day 표시)
    - 레시피 연동 여부 (🍳 아이콘 표시)

### 5) 오늘의 추천 레시피 클릭

- **동작**: 추천 레시피 카드 클릭
- **이동**: 해당 레시피 상세 페이지로 이동
- **데이터**: ramdom으로 3개 레시피 표시

### 6) 상단 검색창 사용

- **입력**: 키워드 입력 (레시피명, 재료명, 공구 제목)

### 7) 로그인 전/후 차이

- **비로그인 상태**:
    - 모든 콘텐츠 조회 가능
    - 참여하기/찜하기/공구 만들기 클릭 시 → 로그인 페이지로 리다이렉트
- **로그인 상태**:
    - 우측 상단 [로그인] → [닉네임 + 프로필 이미지]로 변경
    - 알림 아이콘에 빨간 점 (새 알림 있을 시)
    - 모든 기능 사용 가능
