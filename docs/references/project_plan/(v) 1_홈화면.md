
## 🏠 1. 홈 화면

### 📍 화면 구성

```
┌───────────────────────────────────────────────┐
│          🔥 인기 레시피 캐러셀                   │
│     [← 🍝파스타] [🍛카레] [🥗샐러드 →]          │
├───────────────────────────────────────────────┤
│          ⏰ 마감 임박 공구 배너                  │
│  "🔥 까르보나라 재료 공구 D-1! 서둘러 참여하세요"  │
├───────────────────────────────────────────────┤
│          💡 오늘의 추천 레시피                   │
│         [랜덤 레시피 3개 표시]                   │
└───────────────────────────────────────────────┘

```

### 🎬 사용자 액션 흐름

### 1) 인기 레시피 캐러셀 클릭

- **동작**: 캐러셀 레시피 이미지/제목 클릭
- **이동**: 해당 레시피 상세 페이지로 이동
- **추가**: 좌우 화살표로 슬라이드 전환 (자동 슬라이드 5초 간격)

### 2) 마감 임박 배너 클릭

- **동작**: 배너 전체 영역 클릭
- **이동**: 해당 공구 상세 페이지로 이동
- **표시**: 마감 D-1, D-2 공구만 순환 표시


### 5) 오늘의 추천 레시피 클릭

- **동작**: 추천 레시피 카드 클릭
- **이동**: 해당 레시피 상세 페이지로 이동
- **데이터**: ramdom으로 3개 레시피 표시
