
## ✍️ 6-1. 커뮤니티 글쓰기 페이지

### 📍 화면 구성

```
┌───────────────────────────────────────────────┐
│  커뮤니티 글쓰기 📝                             │
├───────────────────────────────────────────────┤
│  카테고리 *: [꿀팁 공유 ▼]                     │
│                                                │
│  제목 *: [                                  ]  │
│                                                │
│  내용 *:                                       │
│  ┌───────────────────────────────────────┐    │
│  │                                           │    │
│  │                                           │    │
│  │  (텍스트 에디터, 최대 5,000자)             │    │
│  │                                           │    │
│  │                                           │    │
│  └───────────────────────────────────────┘    │
│                                                │
│  이미지: [파일 선택] (최대 5장)                 │
│          [썸네일 미리보기]                      │
│                                                │
│  [레시피 후기인 경우 추가 필드]                 │
│  관련 공구: [검색하여 선택]                     │
│  별점: ⭐⭐⭐⭐⭐ (클릭하여 선택)                │
│                                                │
│  태그: [#식재료보관] [#꿀팁] [추가+]            │
├───────────────────────────────────────────────┤
│     [취소]  [임시저장]  [등록하기] 버튼          │
└───────────────────────────────────────────────┘

```

### 🎬 사용자 액션 흐름

### 1) 카테고리 선택

- **드롭다운**: [꿀팁 공유], [자유 게시판], [레시피 후기]
- **[레시피 후기] 선택 시**: 관련 공구 선택 필드 + 별점 필드 표시

### 2) 제목/내용 입력

- **제목**: 최대 100자
- **내용**: 최대 5,000자, 줄바꿈 가능
- **자동 저장**: 30초마다 브라우저 로컬 스토리지에 임시 저장 (선택 기능)

### 3) 이미지 업로드

- **파일 선택**: 최대 5장, JPG/PNG, 각 5MB
- **미리보기**: 업로드 즉시 썸네일 표시
- **순서 변경**: 드래그로 이미지 순서 변경 가능
- **삭제**: 각 이미지에 [X] 버튼

### 4) 관련 공구 선택 (레시피 후기만)

- **[검색하여 선택] 클릭**:
    - 내가 참여했던 공구 목록 팝업
    - 검색창으로 특정 공구 찾기
- **선택**: 공구 클릭 → 자동 링크 연결
- **표시**: 선택된 공구명 표시

### 5) 별점 선택 (레시피 후기만)

- **별 클릭**: 1~5개 선택
- **hover 효과**: 마우스 올리면 별 색상 변화
- **필수**: 레시피 후기는 별점 필수

### 6) 태그 추가

- **기본 태그**: 카테고리별 자주 쓰는 태그 버튼 제공
- **[추가+] 클릭**:
    - 텍스트 입력 필드 생성
    - #없이 입력 (자동 # 추가)
    - Enter 또는 [추가] 버튼으로 태그 추가
- **태그 삭제**: 각 태그에 [X] 버튼

### 7) [임시저장] 버튼

- **클릭**:
    - 서버에 임시 저장 (status='draft')
    - "임시 저장되었습니다" 알림
    - 마이페이지 > 임시저장 글에서 확인 가능

### 8) [등록하기] 버튼

- **유효성 검사**:
    - 카테고리, 제목, 내용 필수
    - 레시피 후기면 관련 공구 + 별점 필수
- **성공**:
    - 게시글 등록
    - 커뮤니티 해당 탭으로 이동
    - "게시글이 등록되었습니다" 알림

### 9) [취소] 버튼

- **확인 팝업**: "작성 중인 내용이 사라집니다. 취소하시겠습니까?"
- **[임시저장 후 나가기] / [그냥 나가기] / [계속 작성]** 선택
