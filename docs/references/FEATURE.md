
## 📊 11. 기능별 우선순위 정리

### 🔴 1순위 (필수 - Week 1-3)

| 기능 | 설명 | 구현 포인트 |
| --- | --- | --- |
| 회원 관리 | 가입/로그인/프로필 | Spring Security 세션 방식 |
| 공구 CRUD | 일반 공구 등록/조회/수정/삭제 | JPA 엔티티, 이미지 업로드 |
| 참여/취소 | 공구 참여 및 취소 | 중복 방지, 인원 관리 |
| 진행률 표시 | 프로그레스 바 + 상태 관리 | 퍼센트 계산, 색상 변화 |
| 검색/필터 | 제목/내용 검색, 카테고리 필터 | JPA Like 쿼리 |
| 마이페이지 | 내 공구/참여 공구 목록 | 사용자별 데이터 조회 |

### 🟡 2순위 (핵심 차별화 - Week 4)

| 기능 | 설명 | 구현 포인트 |
| --- | --- | --- |
| 레시피 탐색 | TheMealDB API 연동 검색 | RestTemplate, JSON 파싱 |
| 레시피 상세 | 재료/조리법/이미지 표시 | API 데이터 화면 렌더링 |
| 레시피→공구 | 원클릭 공구 생성 (자동 입력) | 재료 파싱 → 공구 Form 자동 채우기 |
| 재료 파싱 | strIngredient1~20 추출 | 반복문으로 재료 목록 생성 |
| 인원별 계산 | 인원 변경 시 1인당 가격 자동 계산 | 간단한 나눗셈 로직 |

### 🟢 3순위 (부가 기능 - Week 5)

| 기능 | 설명 | 구현 포인트 |
| --- | --- | --- |
| 댓글 시스템 | 일반/Q&A 댓글 | Comment 엔티티, type 필드 |
| 후기/매너점수 | 별점 후기 + 온도 반영 | Review 엔티티, 점수 계산 로직 |
| 알림 기능 | 참여/댓글/후기 알림 | Notification 테이블, 상태 변경 시 생성 |
| 찜 기능 | 관심 공구 저장 | Wishlist 테이블 |
| 커뮤니티 | 게시판 기본 CRUD | 별도 Board 엔티티 |
| 식품안전나라 API | 한식 레시피 + 영양 정보 | 선택적 API 연동 |

### ⚪ 4순위 (선택/확장 - Week 6 여유 시)

| 기능 | 설명 | 구현 포인트 |
| --- | --- | --- |
| 번역 API | TheMealDB 영문→한글 | Google Translation API |
| 지도 API | 만남 장소 지도 표시 | 카카오/네이버 지도 |
| 뱃지 시스템 | 조건 달성 시 뱃지 | Badge 테이블, 조건 체크 로직 |
| 포인트 시스템 | 활동별 포인트 적립 | Point 필드, 트리거 |
| 이미지 최적화 | 썸네일 생성, WebP 변환 | Thumbnailator 라이브러리 |
| 실시간 채팅 | 1:1 채팅 (선택) | WebSocket (복잡, 우선순위 낮음) |
