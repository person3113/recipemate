
## 🔍 9. 통합 검색 (상단 검색창)

### 📍 화면 구성 (검색 결과 페이지)

```
┌───────────────────────────────────────────────┐
│  "파스타" 검색 결과                             │
│  [전체] [레시피] [공구] [커뮤니티] 탭            │
├───────────────────────────────────────────────┤
│  [전체 탭 선택 시]                              │
│                                                │
│  📖 레시피 (12)                                 │
│  ┌──────────┐ ┌──────────┐ ┌──────────┐       │
│  │까르보나라 │ │토마토파스타│ │알리오올리오│       │
│  └──────────┘ └──────────┘ └──────────┘       │
│  [더보기]                                       │
│                                                │
│  🛒 공구 (8)                                    │
│  ┌──────────┐ ┌──────────┐                    │
│  │파스타재료 │ │파스타소스 │                    │
│  └──────────┘ └──────────┘                    │
│  [더보기]                                       │
│                                                │
│  💬 커뮤니티 (3)                                │
│  - [꿀팁] 파스타 삶는 법                        │
│  - [후기] 파스타 공구 후기                      │
│  - [자유] 파스타 추천해주세요                   │
│  [더보기]                                       │
└───────────────────────────────────────────────┘

```

### 🎬 사용자 액션 흐름

### 1) 상단 검색창 사용

- **위치**: 모든 페이지 상단 네비게이션
- **입력**: 키워드 입력
- **자동완성**: 입력 중 인기 검색어 드롭다운 표시
    - 최근 검색어 (본인)
    - 인기 검색어 (전체 사용자)
- **Enter 또는 검색 버튼**: 통합 검색 결과 페이지로 이동

### 2) 탭 전환

- **[전체]**: 모든 카테고리 통합 결과 (각 카테고리 상위 3-5개)
- **[레시피]**: 레시피만 전체 결과
- **[공구]**: 공구만 전체 결과
- **[커뮤니티]**: 커뮤니티 게시글만 전체 결과

### 3) 검색 결과 클릭

- **레시피 카드**: 클릭 시 레시피 상세
- **공구 카드**: 클릭 시 공구 상세
- **커뮤니티 항목**: 클릭 시 게시글 상세

### 4) [더보기] 버튼

- **클릭**: 해당 카테고리 탭으로 전환 + 전체 결과 표시

### 5) 검색어 하이라이트

- **표시**: 검색 결과에서 검색어는 노란색 하이라이트
- **예**: "파스타" 검색 → 제목/내용의 "**파스타**" 부분 강조
