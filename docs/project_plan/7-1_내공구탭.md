
## 👤 7-1. [내 공구] 탭

### 📍 화면 구성

```
┌───────────────────────────────────────────────┐
│  내가 만든 공구 (5)                             │
│  [진행중] [완료] [취소] 필터                    │
├───────────────────────────────────────────────┤
│  ┌───────────────────────────────────────┐    │
│  │ 🍝 까르보나라 재료 공구                 │    │
│  │ 상태: 모집중 🟢  |  3/5명  |  D-2       │    │
│  │ [공구 보기] [참여자 관리] [수정] [삭제]  │    │
│  └───────────────────────────────────────┘    │
│  ┌───────────────────────────────────────┐    │
│  │ 🍗 치킨 공동구매                        │    │
│  │ 상태: 완료 ✅  |  10/10명              │    │
│  │ [공구 보기] [후기 보기 (8)]             │    │
│  └───────────────────────────────────────┘    │
│  ...                                          │
└───────────────────────────────────────────────┘

```

### 🎬 사용자 액션 흐름

### 1) 필터 선택

- **[진행중]**: 모집 중인 공구만
- **[완료]**: 마감/목표 달성 공구
- **[취소]**: 삭제된 공구 (참조용)

### 2) [공구 보기] 버튼

- **클릭**: 해당 공구 상세 페이지로 이동

### 3) [참여자 관리] 버튼 (진행중만)

- **클릭**: 참여자 목록 팝업
- **표시 정보**:
    - 참여자 닉네임, 매너온도
    - 참여일, 수령 방법, 수량
- **액션**:
    - [1:1 쪽지] (선택)
    - [강제 탈퇴]

### 4) [수정] 버튼 (진행중만)

- **클릭**: 공구 수정 페이지로 이동

### 5) [삭제] 버튼

- **참여자 확인**:
    - 참여자 있으면 "참여자가 있어 삭제할 수 없습니다"
    - 없으면 "정말 삭제하시겠습니까?" 확인
- **삭제**: 상태를 '취소'로 변경 (완전 삭제 아님)

### 6) [후기 보기] 버튼 (완료만)

- **클릭**: 해당 공구에 달린 후기 목록 팝업
- **표시**: 참여자별 별점 + 텍스트 후기
